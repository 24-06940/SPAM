Getting Started

Welcome to the Preventive Maintenance Management System. This comprehensive system is designed to help engineers and technicians efficiently manage maintenance tasks, equipment procedures, and calibration reports.

System Overview

The Preventive Maintenance Management System consists of several integrated modules:

  • Work Orders - Create, assign, and track maintenance tasks
  • Job Procedures - Standardized maintenance procedures documentation
  • Calibration Reports - Track instrument calibrations and certifications
  • Asset Management - Inventory and status tracking of equipment
  • Reporting & Analytics - Data-driven insights and custom reports
  • User Management - Role-based access control and permissions

Accessing the System

The system is accessible through any web browser. For the best experience, we recommend using the latest version of Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari.

Note: Your system administrator will provide you with login credentials and the appropriate URL to access the system.

Navigation

The main navigation is available through the tabs at the top of each page. The home page provides quick access buttons to critical functions and an overview of the system's key features.

Work Orders Management

The Work Orders module allows you to create, assign, track, and complete maintenance tasks across your organization.

Creating a Work Order

  1. Navigate to the Work Orders tab
  2. Click the "Create Work Order" button
  3. Fill in the required information:
    • Equipment ID - Select the asset requiring maintenance
    • Maintenance Type - Choose from Preventive, Corrective, Predictive, or Emergency
    • Priority - Set as Critical, High, Medium, or Low
    • Description - Detail the work to be performed
    • Assigned To - Email of the responsible technician
    • Due Date - When the work should be completed
  4. Submit the form to create the work order

Filtering Work Orders

Use the filter section at the top of the Work Orders page to narrow down the list by:

  • Status (Open, In Progress, On Hold, Completed)
  • Priority (Critical, High, Medium, Low)
  • Maintenance Type
  • Date Range
  • Assigned Technician

Updating Work Order Status

  1. Find the work order in the list
  2. Click the "Update" button
  3. Select the new status from the dropdown
  4. If marking as "Completed", enter the completion date
  5. Save the changes

Important: Critical and High priority work orders will automatically generate email notifications to responsible personnel.

Job Procedures

The Job Procedures module provides standardized documentation for maintenance tasks to ensure consistency and quality.

Creating a Job Procedure

  1. Navigate to the Job Procedures tab
  2. Click the "Create Procedure" button
  3. Fill in the procedure information:
    • Equipment ID - Associated asset
    • Procedure Title - Descriptive name
    • Safety Protocol - Important safety warnings and requirements
    • Required Tools - List tools and equipment needed
    • Procedure Steps - Step-by-step instructions
    • Expected Outcome - Successful completion criteria
    • Troubleshooting - Common issues and solutions
  4. Submit the form to create the procedure

Viewing and Printing Procedures

To view or print a job procedure:

  1. Find the procedure in the list or use the search function
  2. Click the "View" button to open the detailed view
  3. Use the "Print Procedure" button to generate a printable version

Searching Procedures

Use the search bar at the top of the Job Procedures page to find procedures by name, equipment, or content. You can also filter by equipment using the dropdown filter.

Calibration Reports

The Calibration Reports module helps track instrument calibrations, due dates, and certification status.

Adding a Calibration Report

  1. Navigate to the Calibration Reports tab
  2. Click the "Add Calibration Report" button
  3. Enter the calibration information:
    • Instrument ID - Unique identifier
    • Instrument Name - Descriptive name
    • Calibrated By - Person who performed the calibration
    • Next Calibration Due Date - When recalibration is needed
    • Tolerance Range - Acceptable measurement range
    • Certification Status - Valid, Pending, or Expired
    • Comments - Additional notes or observations
  4. Submit the form to add the calibration report

Tracking Calibration Status

The system automatically tracks calibration due dates and provides:

  • Visual indicators for approaching and overdue calibrations
  • Alerts for instruments due for calibration in the next 7 days
  • Filtering options to view instruments by certification status

Viewing Calibration History

To view the calibration history for an instrument:

  1. Find the instrument in the list
  2. Click the history icon
  3. Review the complete calibration history

Asset Management

The Asset Management module provides a comprehensive inventory of equipment with status tracking and maintenance history.

Registering a New Asset

  1. Navigate to the Asset Management tab
  2. Click the "Register New Asset" button
  3. Enter the asset information:
    • Equipment Name - Descriptive name
    • Model Number - Manufacturer's model
    • Manufacturer - Equipment maker
    • Serial Number - Unique identifier
    • Purchase Date - Acquisition date
    • Location - Physical location
    • Department - Responsible department
    • Status - Active, Maintenance, Inactive, or Out of Service
  4. Submit the form to register the asset

Updating Asset Status

  1. Find the asset in the list
  2. Click the "View" button to open the detailed view
  3. Click "Update Status"
  4. Select the new status and save

Viewing Asset Maintenance History

The detailed view of an asset includes its maintenance history, showing all work orders associated with the equipment.

Creating a Work Order for an Asset

To quickly create a work order for an asset:

  1. Find the asset in the list
  2. Click the "Create Work Order" button
  3. The work order form will open with the equipment ID pre-filled
  4. Complete the remaining information and submit

Reporting & Analytics

The Reporting & Analytics module provides data-driven insights and custom report generation capabilities.

Standard Reports

The following standard reports are available:

  • Work Order Status Report - Summary of work orders by status, priority, and completion rate
  • Calibration Due Report - Instruments due for calibration in a specified date range
  • Asset Performance Report - Equipment maintenance trends and potential issues
  • Department Asset Report - Asset allocation and status by department

Generating Custom Reports

  1. In the Reporting tab, scroll to the "Custom Report Generator" section
  2. Select the report parameters:
    • Report Type - Work Orders, Calibrations, or Assets
    • Date Range - Period to analyze
    • Group By - How to organize the data
    • Format - Spreadsheet or PDF
  3. Click "Generate Custom Report"
  4. The system will generate and display a link to your report

Analytics Dashboard

The analytics section displays visual representations of key performance indicators:

  • Work Order Completion Time - Average days to complete work orders
  • Work Order Status Distribution - Breakdown of current work order statuses
  • Asset Status Overview - Distribution of asset status categories
  • Maintenance Types by Month - Trend of different maintenance activities

User Management

The User Management module controls access to the system through role-based permissions.

User Roles

The system has four defined roles with different permission levels:

  • Administrator - Full access to all system functions and settings
  • Maintenance Manager - Can manage work orders, procedures, assets, and reports
  • Technician - Can view and update work orders assigned to them, view procedures and assets
  • Viewer - Read-only access to work orders, procedures, and assets

Adding a New User

Note: Only Administrators can add new users.

  1. Navigate to the User Management tab
  2. Click the "Add User" button
  3. Enter the user information:
    • Full Name
    • Email Address
    • Role
    • Department
    • Notifications Preference
  4. Submit the form to add the user

Editing User Details

  1. Find the user in the list
  2. Click the "Edit" button
  3. Update the necessary information
  4. Save the changes

Filtering Users

Use the filter controls at the top of the User Management page to filter users by role, department, or search by name or email.
